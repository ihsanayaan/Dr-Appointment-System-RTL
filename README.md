#  Healthcare Booking System

Modern bilingual healthcare booking platform with complete RTL/LTR support built using React.js, Tailwind CSS, Zustand, and i18next.

##  Live Demo
🔗 https://rhombix-technologies-task-3-dr-appo.vercel.app/

##  Figma Design

#  Features

- RTL/LTR bilingual support
- Modern healthcare UI
- Appointment booking system
- Real-time search functionality
- Responsive layouts
- Multi-step booking flow
- Clean user experience
- Mobile-friendly design
- Reusable UI components
- Smooth navigation
